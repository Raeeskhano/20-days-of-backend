✅Introductioin to Express.js:
its a Framwork, and npm pakage.
it manages everything from receiving the request and giving the response.

✅Setting-up a basic Express App:
const express = require("express");

const app = express();

✅Routing: navigatioin to different pages eg-> /, /profile etc
app.get("/", (req, res) => {
  res.send("Hello World");
});

app.get("/profile", (req, res) => {
  res.send("name: Raees khan");
});

const port = 3000;
app.listen(port, () => {
  console.log(`server is running on http://localhost:${port}`);
});
